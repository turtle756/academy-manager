### 해결된 버그
- JWT "Subject must be a string" → sub를 str()로 변환
- SPA fallback → catch-all 라우트로 index.html 반환
- Railway PORT 환경변수 → Dockerfile에서 ${PORT:-8000}
- Docker 프론트엔드 static 경로 → 복수 경로 탐색
- DB address_detail 컬럼 누락 → drop_all 1회 실행 후 제거
- 구글 로그인 콜백 → API JSON 반환 대신 프론트 리디렉트
- AuthProvider 타이밍 → window.location.href로 full reload
