# 학원 올인원 관리 시스템 — 설계 문서

**작성일**: 2026-03-29
**배포**: Railway (단일 배포)
**기술 스택**: FastAPI + React SPA + PostgreSQL (Railway DB)

---

## 1. 프로젝트 개요

학원 운영에 필요한 출결/수납/성적/상담/소통을 하나로 통합한 웹 앱.
학부모는 앱 설치 없이 카카오 알림톡 링크로 조회.
비영리 프로젝트, 수수료 0%.

---

## 2. 사용자 유형

| 유형 | 접근 방식 | 권한 |
|------|----------|------|
| **원장/관리자** | 구글 로그인 → 웹 대시보드 | 전체 (강사 권한 포함) |
| **강사** | 구글 로그인 → 제한된 대시보드 | 자기 수업: 출석/성적/상담 |
| **학부모** | 앱 설치 없음, 알림톡 링크 | 조회만: 출결/성적/납부/공지 |

학생은 별도 계정 없이 원생 데이터로만 존재.

---

## 3. 유저저니맵

### 3-1. 원장 저니 (강사 기능 포함)

```
최초 가입
  구글 로그인 → 학원 정보 입력(이름, 주소, 연락처)
  → 반 생성(중등수학A, 고등영어B...)
  → 시간표 설정(요일/시간/강사/교실)
  → 원생 등록(이름, 연락처, 학부모 연락처, 반 배정)
  → 완료. 출석 받을 준비 끝.

매일 운영
  대시보드 확인 → 오늘 수업 목록 확인
  → 출석 시작 (QR/PIN/수동/키오스크)
  → 출석 현황 실시간 확인
  → 미출석 학생 확인 → 학부모 알림 발송
  → 성적 입력 (시험/과제)
  → 상담 기록 작성 (필요 시)

월말 정산
  수납 현황 확인 → 미납자 알림 발송
  → 교육청 서류 자동 생성 → 다운로드
  → 월간 통계 확인 (출석률, 수납률, 이탈 위험)
```

### 3-2. 강사 저니

```
수업 시작
  로그인 → 오늘 내 수업 확인
  → 출석 시작 (QR/PIN/수동)
  → 수업 진행

수업 후
  성적 입력 → 상담일지 작성 (필요 시) → 끝
```

### 3-3. 학부모 저니

```
알림 수신 (앱 설치 없음)
  카카오 알림톡 수신: "OO이가 3시 5분에 출석했습니다"
  → 링크 클릭 → 웹페이지에서 출결/성적/납부 확인

학원비 납부
  알림톡: "3월 학원비 30만원 청구서"
  → 링크 클릭 → 계좌이체 정보 확인 → 납부
  → 납부 완료 알림
```

## 4. 핵심 플로우

### 4-1. 출석 플로우

```
수업 시간 도래
  │
  ├─ [QR 모드] 강사/원장이 QR 화면 띄움
  │    → 30초마다 QR 자동 갱신
  │    → 학생이 스마트폰으로 스캔
  │    → 서버 검증 (시간 + 위치 옵션)
  │    → 출석 처리 → 학부모 알림톡
  │
  ├─ [PIN 모드] 강사가 4자리 PIN 안내
  │    → 학생이 키오스크/웹에서 PIN 입력
  │    → 서버 검증 → 출석 처리 → 학부모 알림톡
  │
  ├─ [수동 모드] 강사가 명단에서 직접 체크
  │    → 출석/지각/결석/조퇴 선택
  │    → 저장 → 학부모 알림톡
  │
  └─ [키오스크 모드] 학원 입구 태블릿
       → 학생이 이름 선택 or PIN 입력
       → 등원 처리 → 학부모 알림톡

  수업 종료 시
  → 미출석 학생 자동 집계
  → 결석 학부모에게 알림톡 발송
```

### 4-2. 수납 플로우

```
원장이 청구서 생성 (반별/개별)
  → 카카오 알림톡으로 청구서 발송
  → 학부모가 링크로 확인 → 계좌이체 납부 (수수료 0%)
  → 원장이 입금 확인 → 납부 완료 처리
  → 학부모에게 납부 확인 알림톡
  → 미납 시 D-3, D-1, D+1 자동 리마인더
```

### 4-3. 학부모 조회 플로우

```
알림톡 링크 클릭
  → 전화번호 기반 간단 인증
  → 자녀 선택
  → 출결 이력 (캘린더) / 성적 (그래프) / 납부 내역 / 공지사항
```

## 5. UI/UX 화면 구성

### 5-1. 원장 대시보드 (홈)

- 오늘 출석 요약 카드 (출석/전체, 출석률)
- 이번달 출석률 카드
- 미납 현황 카드
- 오늘 수업 일정 리스트 (시간/반/강사/교실/상태)
- 이탈 위험 학생 알림

### 5-2. 출결 관리

- 반 선택 + 날짜 선택
- 출석 모드 선택 탭: QR / PIN / 수동
- QR 모드: QR코드 표시 + 30초 갱신 + 실시간 출석 카운트
- 학생 목록: 출석/지각/결석 상태 + 시간 표시

### 5-3. 키오스크 모드

- 전체 화면, 학원 입구 태블릿용
- PIN 4자리 입력 넘패드
- QR 스캔 안내
- 출석 성공 시 학생 이름 + 반 + 시간 표시 (3초 후 복귀)

### 5-4. 수납 관리

- 월별 청구/납부/미납 요약 + 진행 바
- 청구서 발송 / 미납 리마인더 버튼
- 학생별 납부 상태 리스트

### 5-5. 학부모 조회 (모바일 웹)

- 탭: 출결 / 성적 / 납부
- 출결: 월별 캘린더 (출석/지각/결석 색상), 출석률
- 성적: 과목별 점수 그래프
- 납부: 청구/납부 이력

### 5-6. 강사 화면

- 오늘 내 수업 리스트
- 각 수업별: 출석 시작 / 성적 입력 / 상담일지 버튼
- 최근 상담 메모

### 5-7. 통계

- 출석률 추이 (6개월 라인 차트)
- 반별 출석률 바 차트
- 이탈 위험 학생 TOP 5
- 수납률 비교 (이번달 vs 지난달)

## 6. 네비게이션 구조

```
구글 로그인
  │
  ├─ [원장/관리자] — 좌측 사이드바(PC) / 하단 탭바(모바일)
  │    ├─ 대시보드
  │    ├─ 출결 관리 (반별 현황, QR/PIN/수동, 키오스크)
  │    ├─ 원생 관리 (목록, 등록/수정, 반 배정)
  │    ├─ 시간표 (주간 뷰, 수업 추가/수정)
  │    ├─ 수납 관리 (청구서, 납부 확인, 리마인더)
  │    ├─ 성적 관리 (입력, 학생별 추이)
  │    ├─ 상담일지 (작성, 학생별 이력)
  │    ├─ 공지/소통 (공지 작성, 알림톡 이력)
  │    ├─ 서류 생성 (출석부, 수강생 대장)
  │    ├─ 통계 (출석률, 수납률, 이탈 위험)
  │    └─ 설정 (학원 정보, 강사 관리, 알림톡 연동)
  │
  ├─ [강사]
  │    ├─ 내 수업 (홈)
  │    ├─ 출석 시작
  │    ├─ 성적 입력
  │    └─ 상담일지
  │
  └─ [학부모] (로그인 없음, 알림톡 링크)
       ├─ 전화번호 인증
       ├─ 출결 이력
       ├─ 성적 조회
       ├─ 납부 내역
       └─ 공지사항
```

## 7. 기술 스택

| 영역 | 기술 | 이유 |
|------|------|------|
| 백엔드 | FastAPI (Python) | 빠른 개발, 비동기 지원 |
| 프론트엔드 | React SPA | 동적 UI (QR 갱신, 실시간 대시보드) |
| DB | PostgreSQL (Railway) | 앱과 같은 네트워크, 안정적 |
| 인증 | Google OAuth 2.0 | 간편 로그인 |
| 알림 | 카카오 알림톡 API (비즈엠 등) | 학부모 앱 설치 불필요 |
| 배포 | Railway (단일) | FastAPI가 React 빌드를 static 서빙 |
| 실시간 | WebSocket | 출결 대시보드 실시간 업데이트 |
